Create a new function `populate_monomorphization_registry` in callbacks.rs that:

1. Takes `tcx: TyCtxt<'_>` and `registry: &mut MonomorphizationRegistry` as parameters
2. Iterates over `tcx.hir_body_owners()` — same loop pattern as line 538
3. For each body owner, gets `tcx.optimized_mir(def_id)` — same pattern as line 555
4. Iterates over all basic blocks and their terminators
5. For each `TerminatorKind::Call { func, .. }`:
   a. Resolve the callee DefId from the func operand (use `func.constant()` then `ConstantKind` to get the `FnDef` with DefId and GenericArgs)
   b. Check `tcx.generics_of(callee_def_id).params` — skip if empty (non-generic callee)
   c. Get the generic params from `tcx.generics_of(callee_def_id)` — map each param name
   d. For each generic arg in the call's GenericArgs, resolve to a concrete `rustc_middle::ty::Ty`
   e. If ANY arg is still `ty::TyKind::Param` (unresolvable), skip the whole instantiation with `tracing::debug!("Skipping unresolvable generic instantiation: {} in {}", callee_name, caller_name)`
   f. Convert each resolved ty to `ir::Ty` via `mir_converter::convert_ty`
   g. Build `HashMap<String, Ty>` mapping param names to concrete types
   h. Dedup: check if an identical substitution HashMap already exists for this function before registering
   i. Build label string like `_i32_u64` from the concrete types
   j. Call `registry.register(callee_name, TypeInstantiation::new(substitutions, label))`
