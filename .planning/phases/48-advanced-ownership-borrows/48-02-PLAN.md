    - Test 2: Function with RefCellGhostState + borrow_mut() call when borrow_count>0 generates BorrowConflict VC
    - Test 3: Function with sequential borrow() then borrow() (no borrow_mut) generates no BorrowConflict VC
    - Test 4: borrow() increments borrow_count in SSA state; drop of Ref decrements it
    - Test 5: borrow_mut() sets is_mut_borrowed=true; drop of RefMut clears it
    - Test 6: Function with no RefCellGhostState generates no RefCell VCs (backward compatible)
  </behavior>
  <action>
    1. In vcgen.rs, add `fn generate_refcell_vcs(func: &Function) -> Vec<VerificationCondition>`:
       - For each RefCellGhostState in func.refcell_ghost_states:
         - Declare SMT constants: `(declare-const {borrow_count_var} Int)` and `(declare-const {is_mut_borrowed_var} Bool)`
         - Initialize: `(assert (= {borrow_count_var} 0))` and `(assert (not {is_mut_borrowed_var}))`
       - Walk statements looking for method calls (Terminator::Call) matching RefCell API:
         - `borrow()`: Assert precondition `(not is_mut_borrowed)`, increment borrow_count
         - `borrow_mut()`: Assert precondition `(and (= borrow_count 0) (not is_mut_borrowed))`, set is_mut_borrowed=true
         - `try_borrow()`: Same VC as borrow() (contract violation model, not runtime Result)
         - `try_borrow_mut()`: Same VC as borrow_mut()
         - `into_inner()`: Assert `(and (= borrow_count 0) (not is_mut_borrowed))`
         - `replace()` / `swap()`: Assert `(and (= borrow_count 0) (not is_mut_borrowed))` (requires exclusive access)
       - Each violated precondition generates a VerificationCondition with VcKind::BorrowConflict
       - Use SMT encoding: assert the negation of the safe condition, check-sat; SAT = conflict exists

    2. Call generate_refcell_vcs from the main generate_vcs function, append results to the VC list.

    3. In ir.rs, add helper method `Function::has_refcell_locals(&self) -> bool` returning `!self.refcell_ghost_states.is_empty()`.

    4. Create test file with unit tests constructing Function with RefCellGhostState and verifying correct VC generation for all API methods.
