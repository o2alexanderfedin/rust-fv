# Claude Code Guidelines

## Project Overview

**Toolchain**: Rust
**Goal**: Formal verification addition to Rust

## Development Principles

Follow all guidelines from parent workspace CLAUDE.md with these project-specific additions:

### Rust-Specific
- Follow Rust idioms and best practices
- Leverage Rust's type system for safety
- Write comprehensive tests using `cargo test`
- Use `clippy` for linting
- Format code with `rustfmt`

### Formal Verification
- Mathematical rigor in specifications
- Proof soundness and completeness
- Clear documentation of verification strategies
